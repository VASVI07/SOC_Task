# SOC Analysis Project 🔍

## Overview

This project simulates real-world Security Operations Center (SOC) tasks including log analysis, threat intelligence, alert triage, incident escalation, and evidence handling.

## Objectives

* Understand SOC workflow
* Analyze logs for suspicious activity
* Use threat intelligence concepts
* Perform alert triage
* Document and report incidents

## Project Structure

* 01_Theory_Notes → Basic SOC concepts
* 02_Log_Analysis → Log investigation
* 03_Threat_Intelligence → IP analysis
* 04_Incident_Escalation → Escalation decisions
* 05_Alert_Triage → Alert classification
* 06_Evidence_Preservation → Evidence handling
* 07_Capstone → Final SOC report

## Key Findings

* Multiple failed login attempts detected
* Successful login after failures
* Activity originated from internal IP
* Classified as suspicious and escalated

## Skills Demonstrated

* Log Analysis
* Threat Intelligence
* Incident Response
* Analytical Thinking
* Documentation

## Conclusion

This project demonstrates practical SOC analysis skills and incident handling workflow.
